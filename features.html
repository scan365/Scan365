Four Security Scan Modules

Every scan runs across up to four specialised security modules, each checking a different attack surface of your organisation.

🌐

Website and Domain Security FREE

Checks SSL/TLS certificates, HTTP security headers, Content-Security-Policy, HSTS, weak cipher suites, subdomain takeover risk, directory listing, outdated software, XSS indicators, CORS misconfigurations, API key exposure and OWASP Top 10 web application vulnerabilities.

🎣

Phishing Risk Score FREE

Analyses SPF, DKIM and DMARC email authentication records to determine your organisation's exposure to email spoofing and Business Email Compromise (BEC). Checks for open relay, missing DMARC enforcement, QR-code phishing risk and security awareness training gaps.

☁️

Microsoft 365 and Cloud Audit PRO

Audits your Microsoft 365 tenant for MFA enforcement, Conditional Access policies, legacy authentication, Entra ID configuration, SharePoint external sharing, Teams guest access, Exchange forwarding rules, Microsoft Defender configuration and Privileged Identity Management status.

🛡️

ACSC Essential Eight Assessment PRO

Assesses your organisation against Australia's ACSC Essential Eight security controls from Maturity Level 0 to ML3. Covers application control, patching, Office macro restrictions, user application hardening, admin privilege restriction, OS patching, MFA and backup strategies.

Platform Capabilities

📄

Professional PDF Reports

Every scan generates a branded A4 PDF report with risk score, severity breakdown, module findings, recommendations and IT Service Link footer. Pro reports include all 4 modules with white-label branding.

💬

Aria AI Security Assistant

Aria is your always-on cybersecurity chatbot. Ask questions about your scan results, get remediation guidance, learn about vulnerabilities or connect with the IT Service Link team for expert support.

📊

User Dashboard

Track your security posture over time with a personal dashboard showing scan history, risk score trends, monthly scan usage and quick access to your latest reports.

🔐

MFA-Protected Accounts

All Scan365.io accounts support Multi-Factor Authentication. Password changes require MFA verification, ensuring your scan data and reports remain protected.

🎯

Risk Scoring

Each scan produces an overall risk score from 0 to 100 with colour-coded severity ratings (Low Risk, Medium Risk, High Risk) and individual module scores to help you prioritise remediation.

☁️

Powered by Supabase Sydney

All scan data, user profiles and reports are stored in a PostgreSQL database hosted in Sydney, Australia, ensuring data sovereignty and low latency for Australian organisations.

Framework Alignment

Scan365.io findings are mapped to globally recognised security frameworks used by penetration testers, cloud security engineers and security architects.

🌐 OWASP Top 10

Web application vulnerability classification covering injection, XSS, broken access control, cryptographic failures and more.

🛡️ ACSC Essential Eight

Australian Cyber Security Centre baseline security controls rated at Maturity Levels 0 through 3.

⚔️ MITRE ATT&CK

Adversary tactics and techniques framework used to classify detected threats and attack vectors.

📏 CIS Benchmarks

Center for Internet Security configuration benchmarks for Microsoft 365, Azure and web servers.

🏛️ NIST CSF

NIST Cybersecurity Framework mapping for Identify, Protect, Detect, Respond and Recover domains.

🟦 Microsoft Security Benchmark

Microsoft's Azure and Microsoft 365 security baseline recommendations and hardening guides.
